NB!! change the paths and names

run in terminal to make it executable, the main file: chmod +x /Documents/code/Delta-Robot/Jetson/main.py

move startupLauncher file to the Jetson file and make executable: chmod +x /Documents/code/Delta-Robot/Jetson/startupLauncher.sh


create the file: sudo nano /etc/systemd/system/ros2_boot.service

        [Unit]
        Description=Run Jetson ROS2 Robot on Boot
        After=network.target dev-ttyACM0.device

        [Service]
        ExecStart=/home/jetson/startup_launcher.sh
        User=jetson
        Restart=on-failure
        RestartSec=3

        [Install]
        WantedBy=multi-user.target


enable and start the service: 

        sudo systemctl daemon-reexec
        sudo systemctl daemon-reload
        sudo systemctl enable ros2_boot.service
        sudo systemctl start ros2_boot.service

test: reboot

to see output form the script: journalctl -u ros2_boot.service -n 50 --no-pager

stop the script manually: sudo systemctl stop ros2_boot.service


the script wil run in the background

to disable it: sudo systemctl disable ros2_boot.service
